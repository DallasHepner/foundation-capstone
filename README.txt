- Description -

The purpose of this application is to give a base D&D Character
with race, class, and a name to begin the adventure, with a save function that will screenshot the current page with any edits. Also it has a stats generator that follows the "roll 4d6, dropping the lowest"
method to get the base stats.

There is also a random dice roller for each set of dice to be used,

- Tech Stake -

Javascript, HTML, CSS, PostgreSQL, Node, Express, Axios, Postman

- Features -

Button click for a randomized character for class, race, and a given name

Save button to screenshot the current page
